## 📌 Project Description

The AI Health Prediction System is a web application developed using Streamlit, SQLite, and Google Gemini AI.

It allows users to register patient information, analyze blood report parameters, generate AI-powered health assessments, and securely store patient records for future reference.

The application provides a simple and interactive interface for healthcare data management.

## 🚀 Features

- Register Patient Details
- Blood Report Analysis
- AI Generated Health Assessment
- SQLite Database Integration
- Search Patient Records
- Delete Patient Records
- Export Records to CSV
- Dashboard Statistics
- Responsive User Interface

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/Shrutika0706/AI-Health-Prediction-System.git
```

Go to project folder

```bash
cd AI-Health-Prediction-System
```

Create Virtual Environment

```bash
python -m venv venv
```

Activate

Windows

```bash
venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```text
GEMINI_API_KEY=YOUR_API_KEY
```

Run

```bash
streamlit run app.py
```

## 🔄 Application Workflow

1. Register patient information.

2. Enter blood report values.

3. Click Register Patient.

4. Google Gemini generates an AI health assessment.

5. Patient information is stored in SQLite.

6. Search patient records anytime.

7. Export records to CSV.

8. Delete records when required.
